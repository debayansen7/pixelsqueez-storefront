# PixelSqueez StoreFront 🖼️🤏

Welcome to the **PixelSqueez StoreFront**! This is a modern, fast, and highly responsive landing page and storefront for an image compression service.

## ✨ Features

- **Interactive Image Comparison:** A custom, draggable "Before & After" slider to showcase image compression quality without perceived data loss.
- **Modern Tech Stack:** Built using the modern Next.js App Router, React, and TypeScript for robust type safety.
- **Beautiful Styling:** Fully responsive and styled using Tailwind CSS, featuring a warm "Sunset Cocktail" glassmorphism aesthetic.
- **Centralized Content Management:** All website copy, navigation links, and configuration data are managed centrally in a single file (`src/data/content.ts`) for effortless updates.

## 🚀 Getting Started

To get a local copy up and running, follow these simple steps:

### 1. Install Dependencies

```bash
npm install
```

### 2. Run the Development Server

```bash
npm run dev
```

### 3. View the App

Open http://localhost:3000 with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.
